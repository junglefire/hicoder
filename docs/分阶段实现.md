## 迭代 3：核心代理循环

**目标**：实现完整的代理 Turn 循环 —— 用户提问 → 模型回答/调工具 → 执行工具 → 返回结果 → 循环直到完成。

|--------|------|---------|
| 1.2 | Turn 轮次引擎 | 主循环：构建上下文 → 调模型 → 解析工具调用 → 执行 → 返回结果 → 重复 |
| 1.5 | 上下文构建 | 系统提示词 + 工具定义 + 对话历史拼装 |
| 1.1 | Session 初始化 | 创建会话、加载配置、初始化模型客户端 |

### 核心循环伪码

```python
class Session:
    async def turn(self, user_input: str):
        context = self.build_context(history + user_input)
        async for event in self.model.stream(context):
            if isinstance(event, ToolCall):
                self.pending_tools[event.id] = event
            yield event

        if self.pending_tools:
            results = await asyncio.gather(*[
                self.tool_registry.execute(call)
                for call in self.pending_tools.values()
            ])
            self.history.append_tool_results(results)
            self.pending_tools.clear()
            # 继续下一轮模型调用
            await self.turn("")  # 递归直到模型不再调工具
```

### 验收标准
- 用户输入 → 代理能调用工具并返回最终答案
- 模型同时调多个工具时并发执行
- 用户按 Ctrl+C 可中断当前 turn
- 完整的对话历史维护

### 上下文压缩伪码

```python
MAX_CONTEXT_TOKENS = 128_000

async def maybe_compact(history: list[Message], model: ModelProvider):
    current_tokens = count_tokens(history)
    if current_tokens < MAX_CONTEXT_TOKENS * 0.8:
        return history

    # 保留最近 N 轮对话 + 系统提示
    # 将早期对话压缩为摘要
    summary = await model.stream([
        SystemMessage("Summarize the following conversation concisely..."),
        *history[:-4]  # 保留最近几轮
    ])
    return [SystemMessage(summary)] + history[-4:]
```

### 验收标准
- 对话超过 token 阈值时自动压缩
- 压缩后代理仍能理解上下文
- 钩子脚本可注册并在特定时机执行
- 子代理可 fork 执行子任务并返回结果
- 记忆跨会话持久化
